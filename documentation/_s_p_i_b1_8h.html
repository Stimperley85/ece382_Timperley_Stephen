void SPIB1_Init ( void  )

Initialize SPI B1.

Initialize SPI B1. One feature of the MSP432 is that its SSIs can get their baud clock from either the auxiliary clock (ACLK = REFOCLK/1 = 32,768 Hz see Clock.c) or from the low-speed subsystem master clock (SMCLK <= 12 MHz see Clock.c). The SSI can further divide this clock signal by using the 16-bit Bit Rate Control prescaler Register, UCAxBRW.

Parameters
none
Returns
none
Note
Assumes low-speed subsystem master clock is 12 MHz
