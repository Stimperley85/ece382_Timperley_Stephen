Initialize Timer A1.

Activate Timer A1 interrupts to run user task periodically

Parameters
taskis a pointer to a user function called periodically
periodin 2us units (24/SMCLK), 16 bits
Returns
none
Note
Assumes low-speed subsystem master clock is 12 MHz. With divide by 24 the timer clock will be 500 kHz timer clock. The slowest period is 65535*2us=130ms, or about 8 Hz
