Detailed Description

Provide receive/transmit functions for EUSCI A2.

EUSCI_A2 UART is connected to the PC via the USB cable To be able to use printf
1) add UART2.c to project
2) "..\inc\UART2.h"
3) Must be running at 48 MHz, Clock_Init48MHz();
4) Call UART2_Initprintf()

Remarks
UCA0RXD (VCP receive) connected to P1.2
UCA0TXD (VCP transmit) connected to P1.3
Busy-wait device driver for the EUSCI A0 UART
