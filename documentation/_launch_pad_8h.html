Detailed Description

Provide simple I/O functions for the LaunchPad.

Input from switches, output to LED built-in LED1 connected to P1.0 (not used here) negative logic built-in Button 1 connected to P1.1 negative logic built-in Button 2 connected to P1.4 built-in red LED connected to P2.0 built-in green LED connected to P2.1 built-in blue LED connected to P2.2
