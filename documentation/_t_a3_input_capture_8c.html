Initialize Timer A3 interrupts on P10.4 and P10.5.

Initialize Timer A3 in edge time mode to request interrupts on the rising edges of P10.4 (TA3CCR0) and P10.5 (TA3CCR1). The interrupt service routines acknowledge the interrupt and call a user function.

Parameters
task0is a pointer to a user function called when P10.4 (TA3CCR0) edge occurs
parameter is 16-bit up-counting timer value when P10.4 (TA3CCR0) edge occurred (units of 2/3 usec)
task1is a pointer to a user function called when P10.5 (TA3CCR1) edge occurs
parameter is 16-bit up-counting timer value when P10.5 (TA3CCR1) edge occurred (units of 2/3 usec)
Returns
none
Note
Assumes low-speed subsystem master clock is 12 MHz
